# Phase 3.1.4: Transition UI Components - Design Specification

> **Status:** Design Approved
> **Created:** 2026-03-21
> **Phase:** 3.1.4 (Video Transitions - UI Components)

## Overview

Implement user interface components for transition creation, inspection, and management. This phase provides the visual interaction layer for transitions, building on the foundation models (Phase 3.1.1) and timeline integration (Phase 3.1.2).

## Goals

- **Visual discovery:** Users can browse and discover all available transitions
- **Easy creation:** Multiple intuitive ways to create transitions (drag, menu, prompt, shortcut)
- **Powerful inspection:** Comprehensive inspector for fine-tuning transitions
- **Visual feedback:** Real-time preview of transition effects
- **Workflow integration:** Seamless integration with existing timeline and panels

## Architecture

### Design Philosophy

**Tabbed Panel Approach**
- Share space with existing Effects panel using NSTabViewController
- Keeps related tools accessible without cluttering the UI
- Follows established macOS patterns (Finder sidebar, Xcode inspectors)

**Inspector as Modal Sheet**
- Inspector opens as sheet when transition selected
- Doesn't consume permanent screen space
- Focuses attention on the selected transition
- Three tabs organize complex controls (Properties, Presets, Preview)

**Multiple Creation Methods**
- Different workflows for different user preferences
- Drag-and-drop for visual thinkers
- Context menus for power users
- Auto-prompts for automatic assistance
- Keyboard shortcuts for efficiency

**Timeline Drop Zone:**

```swift
// In TimelineView or ClipTrackView
extension TimelineView: NSDraggingDestination {
    func draggingEntered(_ sender: NSDraggingInfo) -> NSDragOperation {
        guard sender.draggingPasteboard?.availableTypes?.contains(
            TransitionsPaletteViewController.DragPasteboardType.transitionType
        ) ?? false else {
            return []
        }

        return .copy
    }

    func performDragOperation(_ sender: NSDraggingInfo) -> NSDragOperation {
        // Read transition type from pasteboard
        guard let transitionTypeString = sender.draggingPasteboard?.string(
            forType: TransitionsPaletteViewController.DragPasteboardType.transitionType
        ),
              let transitionType = TransitionType(rawValue: transitionTypeString) else {
            return .none
        }

        // Find clips at drop location
        let dropPoint = sender.draggingLocation
        guard let (leadingClip, trailingClip) = findClipsAt(point: dropPoint) else {
            // Show error: "Can only add transitions between two clips"
            return .none
        }

        // Calculate overlap
        let overlapDuration = calculateOverlap(leading: leadingClip, trailing: trailingClip)

        guard overlapDuration >= TransitionValidator.minimumDuration else {
            // Show error: "Clips must overlap by at least 0.1s"
            return .none
        }

        // Create transition
        let transition = TransitionClip(
            type: transitionType,
            duration: min(overlapDuration, CMTime(seconds: 1.0, preferredTimescale: 600)),
            leadingClipID: leadingClip.id,
            trailingClipID: trailingClip.id,
            parameters: TransitionParameters.default(for: transitionType),
            isEnabled: true
        )

        // Add to EditorState
        editorState.addTransition(transition)

        // Open inspector
        editorWindowController?.showTransitionInspector(for: transition.id)

        return .copy
    }
}
```

**Helper Functions:**

```swift
/// Finds the two clips that overlap at a given point in the timeline
/// - Parameter point: Point in timeline coordinate space
/// - Returns: Tuple of (leadingClip, trailingClip) if point is between two overlapping clips, nil otherwise
private func findClipsAt(point: CGPoint) -> (VideoClip, VideoClip)? {
    let timeAtPoint = timelineViewModel.time(at: point)
    let track = timelineViewModel.selectedTrack

    // Find all clips at this time
    let clipsAtTime = track.clips.filter { clip in
        timeAtPoint >= clip.startTime && timeAtPoint <= clip.endTime
    }

    // Need exactly 2 clips for a transition
    guard clipsAtTime.count == 2 else { return nil }

    // Determine leading vs trailing by start time
    let leadingClip = clipsAtTime.min(by: { $0.startTime < $1.startTime })
    let trailingClip = clipsAtTime.max(by: { $0.startTime < $1.startTime })

    guard let leading = leadingClip, let trailing = trailingClip else { return nil }

    // Verify they actually overlap
    let overlapDuration = trailing.endTime - leading.startTime
    guard overlapDuration >= TransitionValidator.minimumDuration else { return nil }

    return (leading, trailing)
}

/// Calculates the overlap duration between two clips
/// - Parameters:
///   - leading: Clip that starts first
///   - trailing: Clip that starts second
/// - Returns: Overlap duration in seconds
private func calculateOverlap(leading: VideoClip, trailing: VideoClip) -> CMTime {
    return max(CMTime(seconds: 0, preferredTimescale: 600), trailing.endTime - leading.startTime)
}
```

```swift
// In ClipTrackView
extension NSView {
    @objc func showAddTransitionMenu(_ sender: Any?) {
        let menu = NSMenu()

        let addTransitionItem = NSMenuItem()
        addTransitionItem.title = "Add Transition"
        addTransitionItem.submenu = createTransitionSubmenu()
        menu.addItem(addTransitionItem)

        // Show menu at cursor location
        let event = sender as? NSEvent
        if let window = window, let view = self as? NSView {
            menu.popUp(positioning: event?.location, in: view)
        }
    }

    private func createTransitionSubmenu() -> NSMenu {
        let menu = NSMenu()

        for preset in BuiltInPresets.presets {
            let item = NSMenuItem()
            item.title = "\(preset.name) (\(formatDuration(preset.duration)))"
            item.target = self
            item.action = #selector(applyPreset(_:))
            item.representedObject = preset
            menu.addItem(item)
        }

        return menu
    }

    @objc private func applyPreset(_ sender: NSMenuItem) {
        guard let preset = sender.representedObject as? TransitionPreset else { return }

        // Get click location in timeline coordinates
        let clickLocation = NSPoint(x: 0, y: 0) // TODO: Extract from menu event

        // Find clips at context location
        guard let (leadingClip, trailingClip) = findClipsAt(point: clickLocation) else {
            showAlert(message: "Can only add transitions between two overlapping clips")
            return
        }

        // Validate overlap
        let overlapDuration = calculateOverlap(leading: leadingClip, trailing: trailingClip)
        guard overlapDuration >= TransitionValidator.minimumDuration else {
            showAlert(message: "Clips must overlap by at least 0.1s")
            return
        }

        // Create transition from preset
        let transition = preset.makeTransition(
            leadingClipID: leadingClip.id,
            trailingClipID: trailingClip.id
        )

        // Add to EditorState
        editorState.addTransition(transition)

        // Open inspector
        editorWindowController?.showTransitionInspector(for: transition.id)
    }
}
```

**Implementation:**

```swift
private func shouldShowAutoPrompt(for overlap: CMTime) -> Bool {
    // Check overlap threshold
    guard overlap >= CMTime(seconds: 0.5, preferredTimescale: 600) else { return false }

    // Check if dismissed recently
    if let dismissed = editorState.lastDismissedAutoPromptDate {
        let minutesSinceDismissal = Date().timeIntervalSince(dismissed) / 60
        guard minutesSinceDismissal >= 5 else { return false }
    }

    return true
}

func onClipDragComplete() {
    // Find all overlaps
    for (leading, trailing, overlap) in findAllOverlaps() {
        guard shouldShowAutoPrompt(for: overlap) else { continue }

        // Check if transition already exists
        if editorState.transitions.contains(where: {
            $0.leadingClipID == leading.id && $0.trailingClipID == trailing.id
        }) {
            continue
        }

        // Show prompt at overlap location
        let position = calculatePromptPosition(leading: leading, trailing: trailing)
        showAutoPrompt(at: position, overlap: overlap)
    }
}
```

```swift
import Foundation
import CoreMedia

/// A preset transition configuration
struct TransitionPreset: Identifiable, Equatable, Codable, Sendable {
    let id: UUID
    var name: String
    let isBuiltIn: Bool
    let transitionType: TransitionType
    let parameters: TransitionParameters
    let duration: CMTime

    init(
        id: UUID = UUID(),
        name: String,
        isBuiltIn: Bool = false,
        transitionType: TransitionType,
        parameters: TransitionParameters,
        duration: CMTime
    ) {
        self.id = id
        self.name = name
        self.isBuiltIn = isBuiltIn
        self.transitionType = transitionType
        self.parameters = parameters
        self.duration = duration
    }

    /// Creates a TransitionClip from this preset
    func makeTransition(
        leadingClipID: UUID,
        trailingClipID: UUID
    ) -> TransitionClip {
        TransitionClip(
            type: transitionType,
            duration: duration,
            leadingClipID: leadingClipID,
            trailingClipID: trailingClipID,
            parameters: parameters,
            isEnabled: true
        )
    }
}
```

## User Workflows

### Workflow 1: Drag-and-Drop Transition Creation

1. User opens Effects & Transitions panel
2. Expands "Basic" category
3. Drags "Quick Dissolve (0.5s)" item
4. Hovers between two overlapping clips in timeline
5. Timeline highlights overlap zone yellow
6. Drops transition item
7. Transition appears in timeline
8. Inspector opens automatically for customization

### Workflow 2: Context Menu Creation

1. User right-clicks on track between two clips
2. Context menu appears with "Add Transition"
3. Submenu shows available transitions with durations
4. User selects "Wipe Left (1.0s)"
5. Transition created immediately
6. Inspector opens

## Error Handling

### Invalid Drop Scenarios

**Not between clips:**
- Show "Can only add transitions between two clips" tooltip
- Cursor changes to forbidden cursor
- Drop operation rejected

**Insufficient overlap:**
- Show "Clips must overlap by at least 0.1s" tooltip
- Highlight overlap region in red
- Drop operation rejected

**Already has transition:**
- Show alert: "Transition already exists between these clips"
- Alert offers three buttons: "Replace", "Keep Both", "Cancel"
- If user clicks "Replace": Remove existing transition, create new one, show inspector
- If user clicks "Keep Both": Abort operation (transitions cannot overlap)
- Preserve custom parameters from old transition in memory (in case user wants to undo)

### Missing Requirements

**If rendering engine doesn't support transition type (shouldn't happen with our 5 types):**
- Show preset with warning icon
- "This transition type requires Metal shaders (not yet available)"
- Disable preset item
